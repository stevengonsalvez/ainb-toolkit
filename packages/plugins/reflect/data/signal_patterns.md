# Signal Detection Patterns (Quick Reference)

Quick-reference version of signal patterns. See `references/signal_patterns.md` for full documentation.

## HIGH Confidence

**Behavioral:**
- Negative: never, don't, stop doing, wrong, incorrect, should not, must not
- Positive: always, must, required, the rule is, correct way
- Frustration: I already told you, again?, not again, how many times

**Knowledge:**
- Root cause: root cause was, the issue was, the problem was, the bug was, caused by
- Fix confirmed: fixed by, the fix was, the solution was, resolved by, solved by
- Decision: chose X over Y, decided to, went with, the tradeoff is
- Breakthrough: turns out, figured out, found it, that worked, it's fixed

## MEDIUM Confidence

**Behavioral:**
- Approval: perfect, exactly, that's right, correct
- Feedback: good, great job, well done, nice, excellent
- Continuation: keep doing, continue with, stick with

**Knowledge:**
- Effort: spent N hours/min, took N to find/fix, after trying, been debugging
- Docs mismatch: the docs say X but, undocumented, misleading error
- Environment: make sure to, requires X installed, only works with/when

## LOW Confidence

**Behavioral:**
- Suggestions: maybe, perhaps, might want to, consider
- Observations: seems like, appears to, looks like

**Knowledge:**
- Implicit success: seems to work, this approach works, so far so good
- Discovery: discovered, found this tool/library, switched to/from
